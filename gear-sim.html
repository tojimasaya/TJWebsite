For Japanese Residents Abroad

海外在住日本人向けオススメ日本SIM

日本の電話番号を維持しながら、海外でSMS認証を受け、日本へ一時帰国したときだけデータを使う。海外在住者にとっての日本SIMは、安さだけでなく「止められない認証」と「帰国時の安心感」で選びます。

1 povo

povo

TJ利用中 / 番号維持 / SMS認証

使わない月は基本料0円で維持でき、海外でも日本のSMSを受けられるので二重認証用として扱いやすい日本SIM。

  • 日本に一時帰国したときだけ必要なデータ量を課金できる。
  • 課金中でなくても海外でSMS受信できるのが大きい。
  • 長期間トッピング購入がないと利用停止・契約解除になる点は注意。
  • 放置すれば自動的に解約されるので、解約手続きの手間が少ない。

2 Rakuten

楽天モバイル

TJ利用中 / 海外2GB / Rakuten Link

3GBまでの低額スタートに海外データ2GBが含まれ、Rakuten Linkを使った通話の強さが海外在住者には大きい。

  • 3GBまで月1,078円（税込）から使える段階制プラン。
  • 海外指定地域では毎月2GBまで追加料金なしで使える。
  • 日本帰国時に使いすぎても段階的に上がるだけなので安心感がある。
  • Rakuten Linkなら海外から日本への通話、日本から海外滞在中の受話でも無料になる場面が多い。
  • Rakuten Linkには発信先・着信方法などの制限があるので事前確認が必要。

3 ahamo

JAL Mobile powered by ahamo

TJ検討中 / JALマイル

月額2,970円で30GB、海外ローミングでも30GB枠を使えるahamo系プラン。JALマイル特典が付くので、次回帰省時に契約を検討中。

  • 30GBを国内外で使えるのは旅行者にはかなり強い。
  • 海外ローミングは利用開始から15日を超えると速度制限（送受信最大128kbps）になる点が海外在住者には重い。
  • 日本に戻って国内通信をするまで海外利用期間がリセットされない。
  • 毎月JALマイルが貯まり、条件次第で「どこかにマイル」に使えるのが魅力。
  • 長期で日本を離れる人は、16日目以降の回避策を別に用意したい。

Global Data Roaming

海外データローミングSIM

日本のキャリアローミングだけに頼らず、旅先で使えるバックアップ回線を別に持っておく。頻繁に国をまたぐ旅行者や、長期滞在中に予備回線を確保したい人向けのSIMです。

香港での購入場所と実名登録について 香港内でプリペイドSIMを購入する場合、各地にある電脳街、コンピュータショップ、コンビニ、旺角の先達廣場などで探せます。香港や中国本土で利用するSIMは、プリペイドSIMを含め、パスポートやIDカードを使った実名登録が必要になるのが基本です。購入場所、物理SIM / eSIM、利用地域、チャージ方法によって手続きや必要書類が変わることがあります。制度や販売条件は変わるため、購入前には必ず各サービスの公式リンクで最新情報を確認してください。
Physical SIM / eSIM

Banana Travel SIM 5G 365 Multi Offer

TJ利用中 物理SIM eSIM対応 52カ国 1年有効

日本、韓国、中国を含む海外52カ国で使えるデータSIM。公式サイトから物理SIMまたはeSIMとして購入でき、1年間有効なので、頻繁に旅行する人や長期滞在者のバックアップ回線として相性が良い。もちろん、日本、韓国、マレーシアといった1カ国のみ、8日間といった短期プランでも購入できます。

Tips: 楽天モバイルは3GBを超えると料金段階が一気に上がるため、海外や一時帰国時にBanana SIM側へデータ通信を逃がして、楽天モバイルを3GB以内に抑えるという使い方もできます。
注意: Banana Travel SIMは香港には対応していません。香港で使う回線として選ぶのではなく、香港以外の対象国へ移動するときのバックアップSIMとして考えます。

ただし「52カ国対応」は万能という意味ではありません。訪問国が対象リストに含まれているか、利用前に公式サイトで必ず確認しておきたいSIMです。

Keep Hong Kong Number

香港から日本に戻る人向け番号維持SIM

香港を離れても、銀行や生活系アプリに登録した香港番号をすぐには捨てられない。日本で香港番号のSMS認証を受け続けたい人に向けた、安価な番号維持用SIMです。

Hong Kong Number / Japan SMS

More Mobile SIM

TJ利用中 物理SIM eSIMあり 香港番号維持 SMS認証

香港から日本に戻る人が、香港の電話番号を安く持ち続けるためのプリペイドSIM。銀行を含む最近のアプリで増えている二重認証用に、日本でも香港番号宛てのSMSを無料で受けられるのが最大のメリットです。

継続したければ追加支払いをすればよく、不要になれば放置して自動解約に任せられる。月額契約で縛られず、必要な間だけ香港番号を残しておけるのが、この用途に向いています。
