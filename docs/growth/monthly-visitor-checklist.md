# Monthly Visitor Growth Checklist

Goal: 1,000 new visitors per month for `tojimasaya.com`.

This checklist is for each GA4 / Search Console review. It keeps the review focused on whether the site is moving toward the goal, and what to change next.

## Baseline

Captured on 2026-06-02, shortly after the Site Guides launch.

- GA4 range: 2026-05-05 to 2026-06-01
- Active users: 35
- New users: 29
- Average engagement time: 3m 27s
- Search Console clicks: 2
- Search Console impressions: 55
- Search Console CTR: 3.6%
- Search Console average position: 9.9
- Sitemap status: submitted and read successfully on 2026-06-02
- New guide URLs: individually indexed, but not yet meaningfully visible in Search Console performance

## North-Star Pace

- Monthly target: 1,000 new users
- Weekly pace: about 250 new users
- Daily pace: about 33 new users

Early checks should not expect this pace immediately. The first milestone is search visibility, then clicks, then repeatable new-user growth.

## Review Cadence

- Monday 09:00 JST: weekly baseline read
- Thursday 09:00 JST: midweek movement check

Use Monday for broader interpretation. Use Thursday to catch early search impressions, indexing changes, or obvious click-through opportunities.

## GA4 Checks

Record:

- New users
- Active users
- First user default channel group
- Sessions by channel: Direct, Organic Search, Organic Social, Referral, Unassigned
- Top landing pages
- Top viewed pages
- Events: page_view, user_engagement, scroll, click_feature_link, open_shirasagi_photo, internal CTA events

Look especially for:

- Organic Search moving above the launch baseline
- Home and Writings sending clicks into the three guide articles
- Guide articles becoming landing pages
- Visitors reaching Gallery, Gear, or Writings after reading a guide

## Search Console Checks

Record:

- Total clicks
- Total impressions
- Average CTR
- Average position
- Top queries
- Top pages
- Queries and pages for:
  - `articles/drone-travel-guide.html`
  - `articles/leica-m2-travel.html`
  - `articles/hong-kong-two-base-life.html`

## Decision Rules

### Wait

Do not change content yet when:

- The sitemap was recently read
- The new guide pages are indexed
- Search Console has not yet accumulated meaningful impressions
- No page has at least 20-30 impressions for the relevant query cluster

### Improve CTR

Revise title or meta description when:

- A page has impressions but CTR stays below about 2%
- Average position is between 3 and 20
- The query clearly matches the page intent

Start with the smallest copy change: title, description, or the first visible guide card text.

### Improve Relevance

Add or adjust headings, FAQ, and internal links when:

- A page has impressions but average position is worse than about 20
- Queries are relevant but broader than the page currently answers
- Search Console shows repeated query themes not clearly covered in the article

### Improve Internal Movement

Adjust Home, Writings, or guide-page links when:

- Guide pages receive views but few visitors continue to another page
- CTA click events stay flat while page views rise
- A guide article becomes a landing page but does not send readers to Gallery, Gear, or related writing

### Add Content

Create a new article only after one of these is true:

- Existing guide articles show impressions for an adjacent topic
- A query cluster appears repeatedly but no current page answers it directly
- One guide article is working and can support a follow-up article through internal links

## Report Format

Each recurring check should return:

1. What changed since the last check
2. Whether the change is meaningful or still too early
3. Which page or query is most promising
4. One recommended action
5. Whether that action should be implemented now or watched until the next check
